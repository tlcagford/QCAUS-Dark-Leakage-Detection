should include and annotate before making strong engineering/security claims.

## How to add/annotate
1. Save PDFs to `literature/papers/` with short filenames (e.g. `yin2017_micius.pdf`).
2. Add a short note in `literature/notes/` describing which parameters the paper constrains (Q factors, coupling limits, link loss, coherence lengths).
3. Keep a `literature/bibliography.bib` for LaTeX/Manuscript usage.
